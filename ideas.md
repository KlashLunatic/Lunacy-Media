# Lunacy Media Website Design Exploration

## Chosen Design: Cosmic Minimalism with Sacred Geometry

**Design Movement:** Modern Luxury meets Cosmic Mythology  
**Inspiration:** A24, FKA Twigs, sacred geometry, digital minimalism

### Core Principles

1. **Mythic Clarity** — Every element serves narrative. No decorative noise; every visual choice reinforces the "sacred book meets digital" aesthetic.
2. **Cosmic Depth** — Dark backgrounds with carefully placed light accents create visual hierarchy and draw focus to content. Gold and purple create a sense of otherworldliness.
3. **Breathing Space** — Generous whitespace and vertical rhythm allow content to resonate. Typography and spacing create rhythm, not density.
4. **Living Documents** — Pages feel like evolving dossiers, not static brochures. Progress indicators, status badges, and update logs reinforce the "build log" narrative.

### Color Philosophy

- **Background:** Deep charcoal (`#07070b`) — cosmic void, professional, timeless
- **Accent Primary:** Purple (`#b478ff`) — mystical, creative energy, calls-to-action
- **Accent Secondary:** Gold (`#d4af37`) — prestige, mythology, highlights
- **Text Hierarchy:** 
  - Primary: Bright white (`#ffffff` at 92% opacity) — clarity
  - Secondary: Muted white (`#ffffff` at 70% opacity) — supporting info
  - Tertiary: Faint white (`#ffffff` at 55% opacity) — metadata, timestamps

**Emotional Intent:** Luxury without coldness. Cosmic without being chaotic. Mythic without being obscure.

### Layout Paradigm

- **Asymmetric Grid System:** 12-column grid with varied column spans (7-5 splits, 4-4-4 cards) to avoid monotony
- **Vertical Rhythm:** Consistent spacing (18px, 26px, 34px, 50px) creates breathing room
- **Hero Section:** Full-width with gradient overlays suggesting cosmic depth
- **Card-Based Content:** Project cards, updates, and CTAs use glassmorphism (semi-transparent with blur) to layer information
- **Sticky Navigation:** Persistent header with blur backdrop maintains context while scrolling

### Signature Elements

1. **Gradient Dot Accent** — Small animated gradient circle (purple → cyan) used as status indicators, list bullets, and visual punctuation
2. **Glassmorphic Cards** — Semi-transparent panels with subtle borders and shadows create depth without heaviness
3. **Status Pills** — Inline badges ("In production," "In progress") with gradient dots communicate project state at a glance

### Interaction Philosophy

- **Hover Elevations:** Buttons and cards lift slightly on hover, creating tactile feedback
- **Smooth Transitions:** All state changes (hover, active, focus) use 150-180ms easing for fluidity
- **Focus States:** Clear keyboard navigation with ring outlines ensures accessibility
- **Loading States:** Gradient animations on buttons during submission

### Animation

- **Entrance:** Cards fade in with subtle upward motion (100ms) as page loads
- **Hover:** Buttons scale slightly and shift color; cards lift 2px with shadow enhancement
- **Transitions:** All color/background changes use 150ms cubic-bezier easing
- **Micro-interactions:** Gradient dots pulse gently; form inputs gain focus rings with glow

### Typography System

- **Display Font:** Cinzel (serif, classical) for headings — reinforces "sacred book" aesthetic
- **Body Font:** Inter (sans-serif, clean) for body text — ensures readability at all sizes
- **Hierarchy:**
  - H1: Cinzel, 48px (clamp to 2.2rem–3.4rem), letter-spacing -0.02em, line-height 1.08
  - H2: Cinzel, 28px, letter-spacing 0.02em
  - H3: Cinzel, 20px
  - Body: Inter, 16px, line-height 1.5
  - Small: Inter, 14px, muted color
  - Kicker: Inter, 12px uppercase, letter-spacing 0.24em

---

## Design Rationale

This approach honors the user's vision of "cosmic, mythic, but clean." By combining:
- **Dark backgrounds** (cosmic void, professional)
- **Purple + gold accents** (luxury, mythology)
- **Generous spacing** (breathing room, clarity)
- **Classical typography** (sacred, timeless)
- **Glassmorphism** (modern, layered)

The result is a site that feels like a living grimoire—a place where creative work is documented, celebrated, and shared with collaborators and funders. Every interaction reinforces the narrative: "We're building something real, and you're invited to watch."
